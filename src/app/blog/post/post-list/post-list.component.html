描述

设计一个 EditText 的文本监听器，停止输入 1s 后，如果文本发生变化则触发监听器。

例子：
文本内容是 111，111 -> 1111 -> 11111，连续输入都小于 1s，在输完后的 1s 触发监听器为 11111；
文本内容是 111，111 -> 1111 -> 111，连续输入都小于 1s，在输完后的 1s 不触发监听器；

类似微信的客户端搜索，不同的是微信在 111 -> 1111 -> 111 是会触发改变的。
